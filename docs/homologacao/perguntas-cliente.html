Como usar este questionário

Este roteiro foi montado a partir da análise do protótipo atual. Hoje o sistema é uma demonstração navegável: muitos fluxos funcionam apenas na tela (botões sem ação, dados de exemplo) e várias integrações externas (fiscal, pagamentos, WhatsApp, Google Drive) ainda são simuladas. Para levar o projeto a produção 100% funcional, é preciso fechar com o cliente as decisões abaixo — cada resposta destrava uma parte do desenvolvimento.

As perguntas marcadas como CRÍTICA são bloqueadoras: sem elas, não é seguro estimar prazo, custo nem começar a construir. Recomenda-se respondê-las primeiro, ainda no kickoff.
